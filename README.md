# Queen's Edict

A CLI tool to play "Queen's Edict" in the Delubrum Reginae (Savage).

![Queen's Edict Demo](./assets/demo.gif)

## Installation

Via cargo:

```
cargo install queens-edict
```

## Usage

Just run the binary and enter your position with the `column row` syntax, e.g.,

```sh
> 1 4
```
